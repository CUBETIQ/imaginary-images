# Imaginary

-   Imginary

### Usages

-   Run (Tempo)

```shell
docker run --rm -p 9000:9000 cubetiq/imaginary:latest
```

-   Run

```shell
docker run -p 9000:9000 \
    --name imaginary \
    --restart always \
    cubetiq/imaginary:latest
    -cors -gzip
```
